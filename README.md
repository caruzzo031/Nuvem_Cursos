#Documentaçaõ do projeto - Site de curso profissionalizantes.

*Introdução- Essse projeto consiste no desenvolvimento de um site voltado a divulgação de  cursos profissionalizantes.

*Objetivo do projeto- Divulgar cursos e  facilitar o acesso à informação.

*Público alvo- Jovens, estudantes e profissionais  em busca de qualificação.

*Tecnologias ultilizadas- HTML e CSS.

*Estrutura do site- Menu, pesquisar e cursos.

*Como executar-  Abrir o arquivo index.html no navegador.

*Conclusão- Projeto de aprendizado.
